# MRS Rules from ZKEEN Dat Files

These files are automatically converted from ZKEEN dat files to MRS format.

## Sources
- GeoIP: https://github.com/jameszeroX/zkeen-ip
- GeoSite: https://github.com/jameszeroX/zkeen-domains

## Format
Files in .mrs format are compatible with:
- [mihomo (Clash Meta)](https://github.com/metacubex/mihomo)
- Other proxy clients supporting MRS format

## Automatic Updates
Conversion runs automatically every 6 hours via GitHub Actions.
